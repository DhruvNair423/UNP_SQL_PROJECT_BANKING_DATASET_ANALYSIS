# UNP_SQL_PROJECT_BANKING_DATASET_ANALYSIS

This project was developed as part of a collaborative learning initiative with UNP Education. The primary objective was to apply fundamental SQL concepts to real-world banking data analysis and database management scenarios. The project involved designing and querying a structured banking database to manage customer information, transactions, and account details. Overall, it provided hands-on experience in using SQL for practical financial data management and analysis.
